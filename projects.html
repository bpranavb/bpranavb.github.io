Project: MitoBERT

MitoBERT is a specialized research project conducted within the Northeastern University AI Club. The goal is to adapt BERT-style transformer architectures—typically used for human language—to the language of biology, specifically mitochondrial DNA. By treating DNA sequences as "text," we can leverage self-attention mechanisms to identify patterns and mutations that might be overlooked by traditional bioinformatics tools. This project requires extensive data preprocessing and an understanding of both genomic structures and transformer weights.

Beyond bioinformatics, I am also developing a drone-based computer vision system. This project utilizes the YOLO (You Only Look Once) model for real-time object detection combined with SLAM (Simultaneous Localization and Mapping) to create detailed 3D maps of environments. The objective is to create a drone that can navigate autonomously while "remembering" objects it has encountered. This work bridges the gap between software-based machine learning and physical hardware execution.
